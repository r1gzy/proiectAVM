Aplicatie web realizata in Streamlit pentru analiza datelor.

Arhiva include: - pagina principala
                - 5 pagini, una pentru a acoperi functionalitatile corespunzatoare fiecarei cerinte
                - un fisier csv cu date de testare, pentru verificarea functionalitatii

Fluxul de utilizare incepe cu incarcarea fisierului csv in aplicatie, odata incarcat, se poate accesa orice pagina din aplicatie pentru a verifica functionalitatea. Daca se intra pe o alta pagina fara a incarca fisierul mai intai, va fi afisat un avertisment.

Fiecare pagina corespunde unei cerinte si acopera functionalitatile cerute in fisierul word.
